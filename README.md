# Todo App

一个使用Python和Tkinter开发的现代化待办事项管理应用。

## 功能特点

- 简洁现代的界面设计
- 任务优先级管理
- 任务分类功能
- 日历视图
- 截止日期提醒
- 自动保存数据
- 搜索功能

## 安装要求

- Python 3.x
- tkcalendar

## 安装步骤

```bash
# 克隆仓库
git clone https://github.com/你的用户名/todo-app.git

# 进入项目目录
cd todo-app

# 安装依赖
pip install tkcalendar
```

## 使用方法

```bash
python todo_app/main.py
```

## 项目结构

```
todo_app/
├── main.py          # 程序入口
├── todo_gui.py      # GUI实现
├── todo_data.py     # 数据管理
└── data/            # 数据存储目录
    └── todos.json
```
